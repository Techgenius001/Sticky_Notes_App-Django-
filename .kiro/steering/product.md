# Product Overview

Midnight Sticky Notes is a dark-themed, minimalist sticky notes application with a free-form drag-and-drop canvas. Users can create, organize, and manage sticky notes on an infinite board with a "Midnight Productivity" aesthetic.

## Core Features

- User authentication (signup, login, logout) using Django's built-in auth
- Dashboard with free-form canvas for note placement
- Sticky notes with:
  - Title, content, color selection, and tags
  - Draggable positioning (x, y coordinates)
  - Resizable dimensions (width, height)
  - Last edited timestamps
- Real-time interactions:
  - Drag notes anywhere on canvas
  - Resize from bottom-right corner
  - Edit via modal dialog
  - Auto-save position and size changes

## Design Philosophy

- Dark theme with "Midnight Productivity" aesthetic (#121212 background)
- Minimalist, distraction-free interface
- Smooth, intuitive interactions
- Quick note creation workflow
